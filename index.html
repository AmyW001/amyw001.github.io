Projects

Book Club

Using React I built a ‘book club’ website that enables users to create an account and sign in, calling data stored in a SQL database using MySQL and Express. The site connects to the Google Books API, and enables users to type in keywords to search for books. The site then displays information from the API call dynamically on the page. Employed Bootstrap and plain CSS for styling.

Pawkies

Worked on a group project, using React to build a platform for dog walkers to add/find dog walks, and live chat with each other. My role was to create a ‘User Profile’ component, which dynamically displays user data called from an SQL database, based on the current URL. I integrated the TalkJS chat API into the page, and created Express routes from the database to generate chat users and enable the chat. Worked with both Bootstrap and pure CSS to style various parts of the website.
